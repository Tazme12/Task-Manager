This is a **full-stack task manager** where the user can **add, update and delete** tasks via a **webpage**.<br>
This **webpage** was built using the **python, flask, html and css**.

## 🛠️ Whats it made of?

- **Frontend**: HTML, CSS, JINJA
- **Backend**: PYTHON, FLASK
- **Database**: SQL

## 🚀 Installation 

1. Clone the repository:<br>
```bash
git clone https://github.com/Tazme12/Full-Stack-Task-Manager.git
cd full-stack-task-manager
```
2. Install the package:<br>
```bash
pip install .
```
## 📌 Usage

Run the CLI app with:<br>
```bash
full-stack-task-manager
```
